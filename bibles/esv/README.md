# ESV Bible

Full text of the English Standard Version (ESV) Bible in Markdown format.

- **Version:** ESV
- **Format:** Markdown (.md)
- **Total verses:** ~31,000
- **License:** Public domain / ESV Used by permission

## Usage

Browse the full text in [`esv.md`](esv.md).

## Format

Each verse is on its own line with the reference in brackets:

```
[Genesis 1:1] In the beginning, God created the heavens and the earth.
```

## About the ESV

The English Standard Version is a word-for-word translation of the Bible, published by Crossway. Used by permission.
